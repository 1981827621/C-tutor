学生提问；老师我的代码有什么问题？代码如下：
#include<bits/stdc++.h>
using namespace std;
const int N=100+5;
int n,a[N];
bool vis[N];//标记某个数有没有被选过
void dfs(int x){
	if (x==n+1){
		for (int i=1;i<=n;i++)cout<<a[i]<<" ";
		cout<<endl;
		return;
	}
	for (int i=1;i<=n;i++){
		if (!vis[i]){//如果这个数没有被选过
			a[x]=i;//第x个位置选这个数
			vis[i]=true;//标记这个数被选过
			dfs(x+1);//选下一个数
		}
	}
}
int main(){
	cin>>n;
	dfs(1);//排列第一个数字
	return 0;
}
老师回答：首先，你要明确你的运行错误原因是超时，然后你要想你代码中哪一部分可能会导致超时，显然只有可能是递归出现了问题。那么你从头开始模拟一下递归的调用过程就会发现你的标记数组出现了问题哦，自己先去模拟一下递归，再去修改一下标记数组试试吧。
