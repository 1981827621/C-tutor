# C++竞赛知识库说明

## 使用方法

开发者请将教学资料文件放在此目录下，支持的格式：
- PDF
- PPT / PPTX
- TXT
- DOCX

## 文件命名建议

- `动态规划基础.pdf`
- `STL容器详解.pptx`
- `图论算法总结.txt`
- `贪心算法技巧.docx`

## 加载说明

1. 将文件放入 `knowledge_base` 目录
2. 启动后端服务时会自动加载所有文档
3. 控制台会显示加载进度和统计信息

## 注意事项

- 文件内容应为纯文本或可提取文本的格式
- 建议每个文件不超过10MB
- 首次启动会下载嵌入模型（约400MB），请耐心等待
